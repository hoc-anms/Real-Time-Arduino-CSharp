With this project my main target: 
  - Arduino
  - Connect circuit to computer via COM port
  - Get data from circuit to computer

### Description

Read data from sensor using arduino and connect to COM port display on computer .

### Tech

My project uses a number of open source projects to work properly:

* [Arduino](http://arduino.cc/) - using languge arduino is microcontroller to communication.
* [Csharp](https://code.visualstudio.com/docs/languages/csharp) - using Csharp windowform to create display show
* [Hardware](https://forum.allaboutcircuits.com/) - design circuit
